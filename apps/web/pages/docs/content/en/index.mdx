# Getting Started with TypeScript

## Introduction
TypeScript is a powerful superset of JavaScript that adds static typing. Let's explore some basic concepts.

### Code Example

```typescript
interface User {
  id: number;
  name: string;
  email?: string; // Optional property
}

// Function with type annotations
function greetUser(user: User): string {
  return `Hello, ${user.name}!`;
}

// Example usage
const newUser: User = {
  id: 1,
  name: "John Doe"
};

console.log(greetUser(newUser)); // Output: Hello, John Doe!
```

## Component Example
You can use React components directly in MDX:

```tsx
import { useState } from 'react'

  const [count, setCount] = useState(0)

  return (
    <button onClick={() => setCount(c => c + 1)}>
      Count: {count}
    </button>
  )
}
```
